TypeScript

  • Type inference

    TypeScript detect the type of value if it is not defined already & can not be assign again with other type value

    TypeScript can store a certain value for a variable for e.g let mySchool:"dps" now mySchool variable wont accept any value except String "dps"

  • Object Type

    define as

    type objectType = {
      name:string,
      age:number
    }

    use as

    let user:objectType = {
      name:"hisham",
      age:25
    }

    it limits number of values

    use as

    let role:[string,string,number,number] = ['admin','manager',1,5]

    define as
